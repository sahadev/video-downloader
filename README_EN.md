# Video Downloader

A Python script to download videos from YouTube, Bilibili, and direct video links.

## Usage

```bash
pip install -r requirements.txt
python download_video.py <video_url>
```

**Examples:**
- YouTube: `python download_video.py https://www.youtube.com/watch?v=VIDEO_ID`
- Bilibili: `python download_video.py https://www.bilibili.com/video/BVxxxxx`
- Direct link: `python download_video.py https://example.com/video.mp4`

Videos are saved to the `downloads/` folder by default.
